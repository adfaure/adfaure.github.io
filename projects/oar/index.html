OAR3

Published 2023-11-10

OAR3 is a platform manager and job orchestrator for computing platform. OAR3 is a python project developed at Université Grenoble Alpes, and originally written by Olivier Richard and Salem Harrache in 2014.

From 2021 to 2023 I have been the main developer of OAR3. My objective was to improve the code base to make it a direct replacement for OAR2. My contributions on the project are:

  • Developing new features.
  • Improving the code base (bug hunting, update libraries, refactoring etc).
  • Maintain the continuous integration pipeline.
  • Continue writing and updating the documentation.
  • Providing support and assisting users.
